# Linux Bootcamp | Filesystem

File System
- control how data is stored and retrieved
- structure & logic rules for managing files
- logical collection of files on a partition or disk

File
- grouping of data

In Linux:
- files and directories are case-sensitive
- most things are considered files (data, devices, sockets, pipes, etc.)
    - if something is not a file, it's a process.

The Linux File System:
- only 1 file tree (unlike Windows which has multiple drive letters & partitions C:/ D:/, etc.)
- follows the Filesystem Hierarchy Standard
    - a hierarchical structure, starting with root `/` directory
