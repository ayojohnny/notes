# Networking | Model - TCP/IP

Network: comprised of many different devices/machines

Network Protocol: a standardized means of communication (i.e. HTTP, IP, UDP, etc.)
- various protocols at different layers
- similar to human languages: English, Spanish, French, etc.

If device A speaks English but device B only speaks French, the devices cannot communicate.

## TCP/IP Model - Layer 1: Network Access
- transmite data over physical media (ethernet)

## TCP/IP Model - Layer 2: Internet
Responibilities:
- connect different networks
- find optimal routes to each network
- deliver packets to destination

Example protocols:
- IPv4
- IPv6
- ICMP (Internet Control Message Protocol)

## TCP/IP Model - Layer 3: Transport Access

Responsibilities
- establish end-to-end communication
- deliver data to destination
    - TCP -> reliable, ordered
    - UDP -> unordered, not-reliable

## TCP/IP Model - Layer 4: Application Access

Responsibilities:
- power various "features" in applications
    - emailing
    - streaming
    - file transfer
    - DNS lookups, etc.
