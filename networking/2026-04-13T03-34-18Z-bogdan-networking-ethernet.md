# Networking | Ethernet

Ethernet: 
- set of standards for data link and physical layers (IEEE 802.3, IEEE 802.11)

802.3:
- standards for physical media (copper cables, fiber cables, etc.)

802.11:
- standards for wireless communication
- 802.11a, 802.11b, 802.11ac, 802.11n are all types of 802.1X stanards
