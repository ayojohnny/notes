# Networking | Networking Model - Open Systems Intercommunication (OSI)

## Open Systems Intercommunication

Layer 1 - Physical
- transmit data over physical media (Ehternet, Wi-Fi, Bluetooth, LTE)

Layer 2 - Data Link
- deals w/ errors on physical layer
- handle flow control
- 2 sublayers:
    - MAC - Media Access Control
    - LLC - Logical Link Control

Layer 3 - Network
- choose best route to destination
- routes packet

Layer 4 - Transport
- data transmission over networking via TCP or UDP

Layer 5 - Session
- responsible for session establishment & termination

Layer 6 - Presentation
- translation, encryption, data compression for application layer
- use formats like JSON etc.

Layer 7 - Application
- the apps users user (gmail, WhatsApp, etc)
