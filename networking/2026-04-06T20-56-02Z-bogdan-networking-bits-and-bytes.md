# Networking | Bits & Bytes

Bit ("binary digit")
- atomic piece of data (smallest non-dividable piece of information)
- either 0 or 1
    - values depend on which media is transporting
        - In fiber: 1 => presence of light, 0 => absence of light
        - In Wi-Fi: 1 => means something else, 0 => means something else

Byte ("octet")
- comprised of 8 bits (i.e. 10001101)


When needing to transmits a character across the network each 
character must be **encoded** based on some encoding standard (i.e. Unicode).

In Unicode, each character can be encoded using 8 bits. For instance `a => 01100001`,
`? => 00111111`, etc.
