# Networking | Ethernet - Media Types

In regards to Ethernet, the data link (L2) is pretty consistent. Major changes in
the networking stack are seen at the physical layer (L1) due to various media types.

Cables - Copper
- Twisted Pair: 8 wires split into 4 pairs (2 twisted together per pair)
    - unshielded twisted pair
    - shielded twisted pair

- Categories determine transfer speeds (higher cat -> faster speeds)
    - 5, 5e, 6, 6a, 7

- RJ-45 (stanard for connectors on copper cables)


Cables - Fiber Optics
- tranfser data via light 
- typically more expensive than cable
- various modes: single mode & multie-mode
    - single mode -> higher transfer rate + longer distances
