# Shell Scripting | Bash

Shell script:
- file containing a series of commands
- plain text file
- top-down execution workflow (simulates typing in commands by hand)

Benefits:
- save time via automation of repetitive tasks
- preserve workflows (i.e. stuff you rarely do, but need to do more than once)
- script can perform faster than a human
- tasks can be delegated to team members & other people on w/in the org.
